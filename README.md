# BRUH

## How to run
Create and activate python environment 
```console
python -m venv env
. env/bin/activate
```
Install all requirements
```console
pip install -r requirements.txt
```
Run
```console
python fit_function.py
```
Example with arguments
```console
python fit_function.py -F LED_2.csv -x V -y mA -f exponential
```
